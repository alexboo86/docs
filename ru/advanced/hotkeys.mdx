Горячие клавиши позволяют выполнять частые действия без использования мыши. Освоение основных сочетаний значительно ускоряет анализ графиков — особенно на быстро движущихся рынках, где каждая секунда на счету.

## Навигация по графику

Сочетания навигации управляют прокруткой, масштабированием и позиционированием на графике.

| Сочетание                     | Действие                                                     |
|-------------------------------|--------------------------------------------------------------|
| Колесо мыши                   | Увеличение / уменьшение масштаба (горизонтальная ось времени) |
| Ctrl + Колесо мыши            | Увеличение / уменьшение масштаба (вертикальная ось цены)     |
| Клик + Перетаскивание (средняя кнопка мыши) | Перемещение графика в любом направлении        |
| Стрелка влево                 | Прокрутка графика влево (назад во времени) на один бар       |
| Стрелка вправо                | Прокрутка графика вправо (вперёд во времени) на один бар     |
| Home                          | Перейти к самым ранним загруженным данным                    |
| End                           | Перейти к последним данным (текущее время)                   |
| Ctrl + 0                      | Сбросить масштаб до уровня по умолчанию                      |

### Поведение прокрутки

Колесо мыши по умолчанию масштабирует горизонтально, расширяя или сжимая ось времени. Удержание **Ctrl** при прокрутке переключает на вертикальное масштабирование, изменяя шкалу цены. Этот двухосевой зум обеспечивает точный контроль над объёмом видимых данных.

<Tip>
Нажмите **End**, чтобы быстро вернуться к текущей рыночной цене после прокрутки исторических данных. Это самый быстрый способ вернуться к режиму реального времени.
</Tip>

## Перекрестие

| Сочетание   | Действие                                                          |
|-------------|-------------------------------------------------------------------|
| C           | Включить/выключить перекрестие                                    |
| Ctrl + C    | Переключение режимов перекрестия (нет, перекрестие, только горизонталь, только вертикаль) |

Перекрестие отображает вертикальную и горизонтальную направляющие линии, следующие за курсором, с метками цены и времени на осях. Режимы перекрестия позволяют выбрать предпочтительный уровень визуальной помощи.

## Инструменты рисования

Сочетания рисования активируют соответствующий инструмент. После активации щёлкните на графике для размещения рисунка. Нажмите **Escape** для отмены текущей операции рисования и возврата к инструменту указателя.

| Сочетание   | Инструмент                                                        |
|-------------|-------------------------------------------------------------------|
| H           | Горизонтальная линия                                              |
| T           | Трендовая линия                                                   |
| R           | Луч (полубесконечная линия)                                       |
| F           | Уровни Фибоначчи                                                 |
| B           | Прямоугольник / Бокс                                              |
| Escape      | Отмена текущего рисунка, возврат к указателю                      |
| Delete      | Удаление выбранного рисунка                                       |

### Процесс рисования

<Steps>
  <Step title="Активируйте инструмент">
    Нажмите клавишу быстрого доступа (например, **H** для горизонтальной линии) или выберите инструмент на панели инструментов.
  </Step>
  <Step title="Разместите рисунок">
    Щёлкните на графике, чтобы установить первую точку привязки. Для многоточечных инструментов (трендовая линия, Фибоначчи) щёлкните ещё раз для установки второй точки.
  </Step>
  <Step title="Скорректируйте при необходимости">
    После размещения щёлкните и перетащите точки привязки для изменения положения. Рисунок привязывается к ценовым уровням для точности.
  </Step>
  <Step title="Деактивируйте инструмент">
    Нажмите **Escape** или щёлкните инструмент указателя на панели инструментов. Рисунок останется на графике.
  </Step>
</Steps>

<Note>
Рисунки сохраняются для каждой вкладки внутри рабочего пространства. Они сохраняются между сессиями и включаются в облачную синхронизацию, если она активирована.
</Note>

## Выделение и редактирование

| Сочетание         | Действие                                                      |
|-------------------|---------------------------------------------------------------|
| Клик по рисунку   | Выделение рисунка (показывает точки привязки)                 |
| Delete            | Удаление текущего выделенного рисунка                         |
| Ctrl + Z          | Отмена последнего действия рисования                          |
| Ctrl + Shift + Z  | Повтор последнего отменённого действия                        |
| Ctrl + A          | Выделение всех рисунков на текущем графике                    |

## Сочетания для индикаторов

| Сочетание   | Действие                                                          |
|-------------|-------------------------------------------------------------------|
| I           | Открыть диалог поиска/добавления индикаторов                      |
| Ctrl + I    | Переключение видимости панели индикаторов (показать/скрыть все)    |

Диалог индикаторов предоставляет список всех доступных индикаторов с возможностью поиска. Начните вводить для фильтрации по названию, затем нажмите **Enter** для добавления выбранного индикатора на активную панель графика.

## Управление рабочим пространством

| Сочетание          | Действие                                                      |
|--------------------|---------------------------------------------------------------|
| Ctrl + N           | Создать новую вкладку в текущем рабочем пространстве          |
| Ctrl + W           | Закрыть активную вкладку                                      |
| Ctrl + Tab         | Переключиться на следующую вкладку                            |
| Ctrl + Shift + Tab | Переключиться на предыдущую вкладку                           |
| Ctrl + 1-9         | Переключиться на вкладку по позиции (1 = первая, 2 = вторая)  |

<Tip>
**Ctrl + Tab** и **Ctrl + Shift + Tab** циклически переключают вкладки, как в браузере. В сочетании с номерными сочетаниями (**Ctrl + 1** ... **Ctrl + 9**) вы можете навигировать по сложным многовкладочным рабочим пространствам без мыши.
</Tip>

## Общие

| Сочетание          | Действие                                                      |
|--------------------|---------------------------------------------------------------|
| F11                | Переключение полноэкранного режима                            |
| Ctrl + ,           | Открыть Настройки                                             |
| Ctrl + S           | Принудительное сохранение текущего состояния рабочего пространства |
| Ctrl + F           | Открыть поиск символа                                         |

## Поиск символа

Нажатие **Ctrl + F** открывает оверлей поиска символов. Введите любую часть названия символа для фильтрации доступных инструментов. Используйте клавиши со стрелками для навигации по результатам и **Enter** для выбора. График в активной панели мгновенно переключается на выбранный символ.

## Настройка

Горячие клавиши в настоящее время предустановлены и не могут быть переназначены через интерфейс. Сочетания разработаны с учётом общепринятых конвенций торговых терминалов и не конфликтуют с системными сочетаниями операционной системы.

<Note>
На macOS замените **Ctrl** на **Cmd** для всех сочетаний. Например, **Cmd + Tab** переключает вкладки, **Cmd + N** создаёт новую вкладку, а **Cmd + ,** открывает Настройки.
</Note>

## Краткая справочная карточка

Для удобства при торговле — наиболее часто используемые сочетания:

| Действие               | Сочетание                   |
|------------------------|-----------------------------|
| Масштаб оси времени    | Колесо мыши                 |
| Масштаб оси цены       | Ctrl + Колесо мыши          |
| Перемещение графика     | Перетаскивание средней кнопкой мыши |
| К последним данным      | End                         |
| Перекрестие             | C                           |
| Горизонтальная линия    | H                           |
| Трендовая линия         | T                           |
| Отмена рисования        | Escape                      |
| Удаление рисунка        | Delete                      |
| Добавить индикатор      | I                           |
| Новая вкладка           | Ctrl + N                    |
| Следующая вкладка       | Ctrl + Tab                  |
| Поиск символа           | Ctrl + F                    |
| Полноэкранный режим     | F11                         |

<Tip>
Распечатайте или сделайте скриншот этой справочной карточки и держите рядом с монитором в процессе обучения. Большинство трейдеров запоминают пять основных сочетаний за несколько дней активного использования.
</Tip>
