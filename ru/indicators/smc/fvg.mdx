Fair Value Gap (FVG) обнаруживает области, где цена двигалась настолько быстро, что между фитилями окружающих свечей остался разрыв. В теории Smart Money Concepts эти разрывы действуют как магниты — институциональные участники ожидают, что цена вернётся и заполнит их перед продолжением движения. Cluster Terminal отображает FVG на текущем таймфрейме и опционально на старших таймфреймах (15M, 1H, 4H, Daily, Weekly), предоставляя многоуровневый обзор незаполненных неэффективностей.

## Как формируются FVG

Fair Value Gap создаётся паттерном из трёх свечей:

1. **Свеча 1** устанавливает опорный максимум или минимум.
2. **Свеча 2** — крупная импульсная свеча, создающая разрыв.
3. **Свеча 3** открывается и торгуется таким образом, что между фитилём Свечи 1 и фитилём Свечи 3 остаётся разрыв.

Разрыв представляет ценовой диапазон, где участвовала только одна сторона рынка. Ожидается, что цена вернётся в эту зону для установления «справедливой стоимости» — позволяя другой стороне совершить сделку.

- **Бычий FVG**: формируется при резком падении цены. Разрыв между минимумом Свечи 1 и максимумом Свечи 3. Ожидается откат цены вверх в разрыв.
- **Медвежий FVG**: формируется при резком ралли цены. Разрыв между максимумом Свечи 1 и минимумом Свечи 3. Ожидается откат цены вниз в разрыв.

<Tip>
FVG старших таймфреймов (4H, Daily, Weekly) имеют значительно больший вес, чем на младших. Незаполненный дневной FVG вблизи ключевого уровня поддержки или сопротивления обеспечивает высоковероятную зону входа.
</Tip>

## Настройки обнаружения

Эти параметры контролируют, как определяются FVG.

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| **lookbackPeriod** | Количество баров для обратного сканирования при обнаружении FVG. Большие значения находят более старые разрывы, но увеличивают нагрузку. | 120 |
| **minGapTicks** | Минимальный размер разрыва в тиках для квалификации как FVG. Установите 0 для обнаружения всех разрывов независимо от размера. Увеличьте на волатильных инструментах для фильтрации шума. | 0 |
| **showInvalidatedFVG** | Продолжать отображать FVG после того, как цена заполнила (повторно посетила) зону. Заполненные зоны отображаются с уменьшенной прозрачностью. | true |

## Настройки старших таймфреймов

FVG со старших таймфреймов проецируются на ваш текущий график, предоставляя контекст от более крупных участников рынка.

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| **showHigherTimeframes** | Главный переключатель для всех FVG старших таймфреймов. | true |
| **show15M** | Показывать 15-минутные FVG. | true |
| **show1H** | Показывать часовые FVG. | true |
| **show4H** | Показывать 4-часовые FVG. | true |
| **showDaily** | Показывать дневные FVG. | true |
| **showWeekly** | Показывать недельные FVG. | true |

<Note>
FVG старших таймфреймов рассчитываются индикатором внутри — вам не нужно переключать таймфрейм графика. Индикатор агрегирует свечные данные для построения HTF-свечей и обнаружения их разрывов.
</Note>

## Визуальные настройки

Управление внешним видом зон FVG на графике.

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| **bullishColor** | Цвет заливки бычьих зон FVG. | #22c55e55 |
| **bearishColor** | Цвет заливки медвежьих зон FVG. | #ef444455 |
| **borderColor** | Цвет границы всех зон FVG. | #9ca3af |
| **borderWidth** | Толщина границы в пикселях. | 1 |
| **showLabels** | Отображать текстовые метки на зонах FVG с указанием направления и таймфрейма. | true |
| **labelColor** | Цвет текста меток. | #000000 |
| **labelFontSize** | Размер шрифта меток в пикселях. | 10 |
| **showMidline** | Нарисовать горизонтальную центральную линию на уровне 50% каждой зоны FVG. Средняя линия часто выступает как точная точка реакции. | false |
| **unifiedStyle** | Использовать одинаковые бычьи/медвежьи цвета для всех таймфреймов вместо индивидуальных цветов. | false |

|----------|----------|--------------|
| **bullish15MColor** / **bearish15MColor** | Цвета 15-минутных FVG | по теме |
| **bullish1HColor** / **bearish1HColor** | Цвета часовых FVG | по теме |

| **bullishWeeklyColor** / **bearishWeeklyColor** | Цвета недельных FVG | по теме |

## Настройки оповещений

Оповещения FVG срабатывают, когда цена приближается к незаполненному разрыву, давая вам заблаговременное предупреждение перед потенциальной реакцией.

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| **alertDistance** | Расстояние в тиках от границы FVG, при котором срабатывает оповещение. | 5 |
| **alert15M** | Включить оповещения для 15-минутных FVG. | true |
| **alert1H** | Включить оповещения для часовых FVG. | true |
| **alert4H** | Включить оповещения для 4-часовых FVG. | true |
| **alertDaily** | Включить оповещения для дневных FVG. | true |
| **alertWeekly** | Включить оповещения для недельных FVG. | true |

### События оповещений

| Event | Срабатывает когда |
|-------|-------------------|
| **zone_approached** | Цена приближается на `alertDistance` тиков к границе незаполненного FVG. |

Настройте звуки и уведомления в глобальной панели настроек оповещений.

## Практическое применение

### Иерархия качества FVG

Не все FVG равноценны. Качество зависит от нескольких факторов:

| Фактор | Более высокое качество | Более низкое качество |
|--------|----------------------|----------------------|
| **Таймфрейм** | Daily, Weekly | 1M, 5M |
| **Размер разрыва** | Большой относительно ATR | Крошечные разрывы |
| **Контекст** | На ключевых уровнях S/R | В середине диапазона |
| **Статус заполнения** | Незаполненный (свежий) | Частично заполненный |
| **Сила импульса** | Сильная импульсная свеча | Слабое движение |

### Конфлюэнтные установки

FVG наиболее эффективны в сочетании с другими инструментами:

- **Order Blocks + FVG**: FVG, перекрывающийся с ордер-блоком, создаёт зону высокой конфлюэнции, где совпадают ценовая неэффективность и институциональное позиционирование.
- **Volume Profile POC**: FVG вблизи POC профиля объёма указывает, что рынок считает этот уровень важным как с временной, так и со структурной точек зрения.
- **Market Structure (BOS/CHoCH)**: FVG, сформированные во время структурного пробоя, более значимы, поскольку представляют неэффективность, созданную самим пробоем.

### Распространённые ошибки

- **Торговля каждого FVG**: FVG младших таймфреймов заполняются постоянно. Фокусируйтесь на HTF-разрывах или тех, что расположены у структурных уровней.
- **Ожидание точного заполнения**: цена не всегда полностью заполняет FVG. Средней линии (50%) часто достаточно для удовлетворения неэффективности.
- **Игнорирование контекста тренда**: FVG в направлении тренда с большей вероятностью удержатся. Контртрендовые FVG заполняются легче.

<Warning>
FVG — это структурные зоны, а не торговые сигналы сами по себе. Всегда подтверждайте данными потока ордеров (дельта, поглощение, имбалансы) перед входом в позицию на уровне FVG.
</Warning>
