Это руководство проведёт вас через интерфейс Cluster Terminal — от создания
первого рабочего пространства до размещения индикаторов и инструментов
рисования на графике. К концу у вас будет полностью настроенная аналитическая
среда, готовая к работе с реальными данными.

## Обзор интерфейса

Cluster Terminal использует макет на основе рабочих пространств. Каждое
рабочее пространство содержит одну или несколько **панелей**, а каждая панель
отображает график, панель индикаторов или виджет. Основные области интерфейса:

{/* screenshot: Full application window with labeled areas — toolbar, tab bar, chart area, status bar */}

| Область | Расположение | Назначение |
|---------|-------------|-----------|
| **Панель инструментов** | Верх каждой панели | Выбор символа, таймфрейма, типа графика, меню индикаторов и рисования |
| **Панель вкладок** | Верх рабочего пространства | Переключение между открытыми вкладками; нажмите **+** для добавления новой |
| **Область графика** | Центр | Основная поверхность отрисовки свечей, кластеров, тепловой карты или TPO-профилей |
| **Строка состояния** | Низ окна | Статус подключения, задержка сервера и текущее состояние данных |
| **Панели** | Сбоку или поверх | Список индикаторов, панель инструментов рисования, виджет DOM, настройки |

<Tip>
  Каждый элемент панели инструментов контекстно привязан к активной панели. Если
  у вас открыто несколько панелей в разделённом макете, нажатие на панель делает
  её активной и обновляет панель инструментов соответственно.
</Tip>

## Создание первого рабочего пространства

Рабочее пространство — это именованный контейнер, сохраняющий всю вашу
аналитическую конфигурацию: открытые графики, выбранные символы, таймфреймы,
индикаторы, рисунки и расположение панелей. Вы можете создать сколько угодно
рабочих пространств — одно для скальпинга, другое для свинг-анализа, третье
для мониторинга альткоинов.

<Steps>
  <Step title="Откройте диалог рабочих пространств">
    При первом запуске (или из меню) появляется **Диалог рабочих пространств**.
    Он отображает список существующих рабочих пространств и кнопку для создания нового.

    {/* screenshot: Workspace dialog with "Create new workspace" button highlighted */}
  </Step>

  <Step title="Назовите рабочее пространство">
    Нажмите **Создать новое рабочее пространство**. Введите описательное название —
    например, «BTC Скальпинг 5м» или «Альткоины Свинг». Название поможет
    идентифицировать пространство при переключении между настройками.
  </Step>

  <Step title="Подтвердите создание">
    Нажмите **Создать**. Новое рабочее пространство откроется с одной пустой
    вкладкой, готовой к настройке.
  </Step>
</Steps>

<Note>
  Рабочие пространства сохраняются локально на вашем устройстве. Если ваша
  лицензия включает функцию **Облачной синхронизации**, рабочие пространства
  также автоматически синхронизируются между устройствами. Подробнее см.
  [Облачная синхронизация](/workspace/cloud-sync).
</Note>

## Выбор биржи

Cluster Terminal подключается к данным централизованных бирж через свой
серверный бэкенд. Поддерживаемые биржи — **Binance Futures** (бессрочные
контракты USDT-M) и **Binance Spot**, обеспечивающие наибольшую ликвидность
и наиболее полные кластерные данные.

При добавлении графика или смене символа вы увидите селектор биржи в панели
инструментов. Выберите **Binance Futures** или **Binance Spot** для доступа
к доступным торговым парам.

<Note>
  Поддержка дополнительных бирж (например, Bybit) запланирована в будущих
  релизах. Селектор бирж спроектирован для подключения нескольких источников по
  мере их доступности.
</Note>

## Выбор символа и таймфрейма

<Steps>
  <Step title="Откройте селектор символов">
    Нажмите на **название символа** в панели инструментов (или на значок лупы),
    чтобы открыть панель поиска символов. Вы увидите список доступных торговых
    пар, отсортированных по объёму.

    {/* screenshot: Symbol selector dropdown showing BTCUSDT, ETHUSDT, SOLUSDT, etc. */}
  </Step>

  <Step title="Поиск или просмотр">
    Введите название символа (например, «BTC» или «ETH»), чтобы отфильтровать
    список. Нажмите на нужную пару для загрузки. Популярные варианты для начала:

    | Символ | Описание |
    |--------|----------|
    | **BTCUSDT** | Бессрочный Bitcoin — максимальная ликвидность, лучшие кластерные данные |
    | **ETHUSDT** | Бессрочный Ethereum — вторая по ликвидности пара |
    | **SOLUSDT** | Бессрочный Solana — высокая волатильность, активный поток ордеров |

  </Step>

  <Step title="Выберите таймфрейм">
    Используйте **селектор таймфрейма** в панели инструментов для выбора
    интервала анализа. Доступные таймфреймы:

    `1m` · `3m` · `5m` · `15m` · `30m` · `1h` · `2h` · `4h` · `6h` · `12h` · `1D`

    Короткие таймфреймы (1m–5m) обеспечивают детальные кластерные данные, идеальные
    для скальпинга. Длинные таймфреймы (1h–1D) агрегируют объём для свинг- и
    позиционного анализа.

    {/* screenshot: Timeframe selector buttons in the toolbar */}
  </Step>
</Steps>

<Tip>
  Хорошая отправная точка — **BTCUSDT на 5-минутном таймфрейме** в режиме
  кластеров. Эта комбинация даёт достаточно деталей для просмотра отдельных
  уровней объёма, сохраняя читаемость графика.
</Tip>

## Типы графиков

Cluster Terminal предлагает четыре различных режима визуализации графиков.
Переключайтесь между ними с помощью переключателя типа графика в панели инструментов.

### Свечи (Candlestick)

Классический свечной график OHLCV. Каждая свеча показывает цены открытия,
максимума, минимума и закрытия за выбранный таймфрейм. Объём отображается
как гистограмма под графиком.

{/* screenshot: Standard candlestick chart with volume histogram */}

**Лучше всего подходит для:** общего обзора рынка, определения тренда, уровней
поддержки и сопротивления.

### Кластеры (объёмный профиль по свече)

Фирменный вид Cluster Terminal. Каждая свеча раскладывается в вертикальный
объёмный профиль, показывающий **объём покупок и продаж на каждом ценовом
уровне** внутри диапазона свечи. Интенсивность цвета указывает на
концентрацию объёма.

{/* screenshot: Cluster chart showing per-level buy/sell volume with color gradient */}

**Лучше всего подходит для:** выявления абсорбции, инициативных покупок/продаж,
дисбалансов объёма и точных уровней входа/выхода.

### Тепловая карта

Непрерывная цветовая визуализация объёма по времени и цене. Плотные участки
отображаются яркими пятнами, позволяя легко определить зоны высокой активности
без дискретных границ свечей.

{/* screenshot: Heatmap view with bright volume concentration zones */}

**Лучше всего подходит для:** визуализации плотности объёма за длительные
периоды, обнаружения повторяющихся зон поддержки/сопротивления и выявления
паттернов институциональной активности.

### TPO (Time Price Opportunity)

График рыночного профиля, показывающий, сколько **времени** цена провела на
каждом уровне, отображённый в виде сложенных букв (блоков TPO). Это выявляет
зону стоимости, точку контроля и форму распределения сессии.

{/* screenshot: TPO profile chart with value area highlighted */}

**Лучше всего подходит для:** определения зон стоимости, начального баланса
и одиночных отпечатков. Полезен для понимания рыночной структуры и теории аукциона.

<Note>
  Все четыре типа графиков используют одни и те же базовые данные и
  переключаются мгновенно без перезагрузки. Индикаторы и рисунки сохраняются
  при переключении между режимами, хотя некоторые визуальные элементы могут
  отображаться по-разному в зависимости от типа графика.
</Note>

## Добавление первого индикатора

Индикаторы предоставляют дополнительные аналитические слои поверх необработанных
данных о цене и объёме. Cluster Terminal включает библиотеку классических
индикаторов, индикаторов потока ордеров и концепций умных денег (SMC).

<Steps>
  <Step title="Откройте панель индикаторов">
    Нажмите кнопку **Индикаторы** (значок графика) в панели инструментов, чтобы
    открыть панель выбора индикаторов.

    {/* screenshot: Indicators button in toolbar highlighted */}
  </Step>

  <Step title="Просмотрите категории">
    Индикаторы организованы по категориям:

    | Категория | Примеры |
    |-----------|---------|
    | **Классические** | EMA, SMA, Bollinger Bands, RSI, MACD |
    | **Поток ордеров** | CVD (Кумулятивная дельта объёма), OI Delta, Volume Delta |
    | **Метрики** | Ставка финансирования, ликвидации, соотношение Long/Short |
    | **Профили** | Volume Profile, VWAP, Session VWAP |
    | **SMC** | Order Blocks, Fair Value Gaps, Break of Structure |

  </Step>

  <Step title="Добавьте индикатор">
    Нажмите на название индикатора, чтобы добавить его на активный график.
    Некоторые индикаторы отображаются как оверлеи на ценовом графике (например,
    EMA или VWAP), другие — в отдельной подпанели под графиком (например,
    RSI или CVD).
  </Step>

  <Step title="Настройте параметры">
    После добавления нажмите **значок шестерёнки** рядом с названием индикатора,
    чтобы открыть его настройки. Настройте параметры: период, цвета, стиль
    линии и видимость. Изменения применяются мгновенно.

    {/* screenshot: Indicator settings popover with period, color, and style options */}
  </Step>
</Steps>

<Tip>
  Для первой настройки попробуйте добавить **EMA 21** и **EMA 50** как оверлеи,
  а также **CVD** (Кумулятивная дельта объёма) как подпанель. Это сразу даст вам
  контекст тренда и обнаружение дивергенций потока ордеров.
</Tip>

## Использование инструментов рисования

Инструменты рисования позволяют аннотировать графики трендовыми линиями,
горизонтальными уровнями, прямоугольниками, уровнями Фибоначчи и другими
элементами.

<Steps>
  <Step title="Откройте панель инструментов рисования">
    Нажмите кнопку **Инструменты рисования** (значок карандаша) в панели
    инструментов. Появится панель со списком всех доступных инструментов.

    {/* screenshot: Drawing tools panel with categories: lines, shapes, Fibonacci, text */}
  </Step>

  <Step title="Выберите инструмент">
    Нажмите на нужный инструмент — например, **Трендовая линия** или
    **Горизонтальный луч**. Курсор изменится на перекрестие, указывая, что
    режим рисования активен.
  </Step>

  <Step title="Нарисуйте на графике">
    Нажмите на графике, чтобы установить начальную точку, затем нажмите снова
    для установки конечной точки (для двухточечных инструментов, таких как
    трендовые линии). Для одноточечных инструментов, таких как горизонтальные
    уровни, одно нажатие размещает рисунок.
  </Step>

  <Step title="Настройте и стилизуйте">
    Нажмите на любой существующий рисунок, чтобы выбрать его. Перетаскивайте
    контрольные точки для изменения положения. Правый клик открывает опции:
    цвет, толщина линии, продление до бесконечности и удаление.
  </Step>
</Steps>

<Note>
  Рисунки сохраняются вместе с рабочим пространством. Они сохраняются между
  сессиями и синхронизируются при включённой облачной синхронизации.
</Note>

## Сохранение рабочего пространства

Cluster Terminal автоматически сохраняет состояние рабочего пространства в
процессе работы. Тем не менее полезно понимать, что и когда сохраняется:

- **Автосохранение** — изменения макета, индикаторов и рисунков сохраняются
  непрерывно в локальное хранилище.
- **Ручное сохранение** — вы можете запустить сохранение вручную из диалога
  рабочих пространств или с помощью горячей клавиши.
- **Облачная синхронизация** — если включена, рабочее пространство загружается
  на сервер периодически (каждые 5 минут) и при закрытии приложения.

{/* screenshot: Workspace dialog showing saved workspaces with cloud sync badges */}

<Tip>
  Используйте **Диалог рабочих пространств** (доступен из меню или панели
  инструментов) для переименования, дублирования или удаления рабочих
  пространств. Диалог также отображает значок облачной синхронизации,
  показывающий, синхронизировано ли каждое рабочее пространство.
</Tip>

## Вкладки и разделённые макеты

Вы можете открыть несколько графиков в одном рабочем пространстве, используя
вкладки и разделённые панели. Это удобно для одновременного мониторинга
нескольких символов или таймфреймов.

### Работа с вкладками

- **Добавить вкладку** — нажмите кнопку **+** в конце панели вкладок. Каждая
  новая вкладка начинается как пустой график, который можно настроить независимо.
- **Переключить вкладки** — нажмите на любую вкладку, чтобы вывести её на передний план.
- **Закрыть вкладку** — нажмите значок **x** на вкладке, чтобы удалить её из
  рабочего пространства. Данные при этом не удаляются.
- **Переупорядочить вкладки** — перетаскивайте вкладки для изменения их порядка.

### Разделение панелей

Вы можете разделить область графика горизонтально или вертикально для просмотра
нескольких графиков рядом в одной вкладке:

- **Перетащите вкладку** к краю области графика (верх, низ, лево или право),
  чтобы создать разделённую панель. Синяя подсветка указывает, где появится
  новая панель.
- **Измените размер панелей**, перетаскивая разделитель между ними.
- Каждая панель имеет собственную независимую панель инструментов, символ,
  таймфрейм и конфигурацию индикаторов.

{/* screenshot: Two-pane split layout showing BTCUSDT cluster chart and ETHUSDT candle chart */}

<Tip>
  Популярная многопанельная настройка: кластерный график на 5м в основной панели,
  свечной график на 1ч в меньшей боковой панели для контекста старшего таймфрейма
  и виджет DOM в узкой колонке справа.
</Tip>

## Горячие клавиши

Cluster Terminal поддерживает горячие клавиши для типичных действий. Вот
основные для начала работы:

| Сочетание | Действие |
|-----------|---------|
| `Ctrl + N` | Новая вкладка |
| `Ctrl + W` | Закрыть текущую вкладку |
| `Ctrl + S` | Сохранить рабочее пространство |
| `Ctrl + Z` | Отменить последнее действие рисования |
| `Ctrl + Shift + Z` | Повторить последнее действие рисования |
| `Escape` | Отменить текущий инструмент рисования / снять выделение |
| `Delete` | Удалить выбранный рисунок |
| Колесо мыши | Масштабирование по оси времени |
| Перетаскивание средней кнопкой | Перемещение графика |

Полный список сочетаний и параметры настройки см. в разделе
[Горячие клавиши](/advanced/hotkeys).
