## Обзор

**Уровни ликвидаций** рисует горизонтальные линии на ценовом графике на конкретных предполагаемых ценах, где сработают ликвидации, сгруппированных по тирам плеча. В отличие от Карты ликвидаций (которая показывает зоны плотности), этот индикатор показывает дискретные ценовые уровни, что упрощает установку алертов и определение точных целей.

Поддерживаются три тира плеча:
- **Низкое плечо** (1-20x) — более крупные позиции, широкие стоп-расстояния, сложнее спровоцировать.
- **Среднее плечо** (20-50x) — типичное розничное плечо, умеренная чувствительность.
- **Высокое плечо** (50-200x) — высокомаржинальные позиции, очень чувствительны к движению цены.

## Как это работает

Индикатор оценивает цены ликвидации на основе распределения открытого интереса и распространённых уровней плеча в каждом тире. Линии рисуются на ценах, где предполагается значительный объём ликвидаций.

Каждый тир имеет независимое визуальное оформление (цвет, толщина линии, паттерн штриховки) и может быть включён или выключен как для лонгов, так и для шортов.

## Тип графика

| Свойство | Значение |
|----------|----------|
| Тип | Оверлей (горизонтальные линии на основном графике) |
| Положение | Основной ценовой график |
| Отображение | Стилизованные горизонтальные линии с опциональным свечением |

## Настройки

### Отображение

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| displayMode | Стиль отображения | `lines` |
| lineVisual | Пресет стиля линий | `neon` |
| volumeGradation | Масштабировать прозрачность/толщину линии по предполагаемому объёму | `true` |
| lookbackBars | Диапазон истории для расчёта (0 = все видимые) | `0` |

### Диапазоны тиров плеча

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| leverageLowFrom | Минимальное плечо низкого тира | `1` |
| leverageLowTo | Максимальное плечо низкого тира | `20` |
| leverageMediumFrom | Минимальное плечо среднего тира | `20` |
| leverageMediumTo | Максимальное плечо среднего тира | `50` |
| leverageHighFrom | Минимальное плечо высокого тира | `50` |
| leverageHighTo | Максимальное плечо высокого тира | `200` |

### Переключатели видимости

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| showLongsLow | Показывать уровни ликвидации лонгов (низкий тир) | `true` |
| showLongsMed | Показывать уровни ликвидации лонгов (средний тир) | `true` |
| showLongsHigh | Показывать уровни ликвидации лонгов (высокий тир) | `true` |
| showShortsLow | Показывать уровни ликвидации шортов (низкий тир) | `true` |
| showShortsMed | Показывать уровни ликвидации шортов (средний тир) | `true` |
| showShortsHigh | Показывать уровни ликвидации шортов (высокий тир) | `true` |

### Стиль линий

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| lowWidth | Толщина линий низкого тира плеча | `1` |
| medWidth | Толщина линий среднего тира плеча | `1.5` |
| highWidth | Толщина линий высокого тира плеча | `2` |
| lowDash | Стиль линий низкого тира плеча | `dot` |
| medDash | Стиль линий среднего тира плеча | `dash` |
| highDash | Стиль линий высокого тира плеча | `solid` |

### Эффекты свечения

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| enableGlow | Включить эффект свечения на линиях | `false` |
| glowIntensity | Яркость свечения (0-100) | `50` |
| glowLayers | Количество слоёв свечения | `2` |

### Фильтры по размеру

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| showSizeTier1 | Показывать ликвидации мелких позиций | `true` |
| showSizeTier2 | Показывать ликвидации средних позиций | `true` |
| showSizeTier3 | Показывать ликвидации крупных позиций | `true` |

## Премиальные стили линий

Уровни ликвидаций поддерживают премиальные визуальные стили для улучшенной эстетики графика:

| lineVisual | Описание |
|------------|----------|
| `flat` | Сплошной цвет, без эффектов. Чистый и минималистичный. |
| `neon` | Светящийся неоновый эффект с настраиваемой интенсивностью (по умолчанию). |
| `gradient` | Цветовой градиент вдоль длины линии. |
| `chrome` | Металлический отражающий вид. |

Когда `enableGlow` установлен в `true`, все стили получают дополнительный эффект внешнего свечения, контролируемый параметрами `glowIntensity` и `glowLayers`.

## Интерпретация

### Значимость тиров плеча

<Tip>
Уровни ликвидации высокого плеча (50x+) являются наиболее магнетическими — цена часто охотится за этими уровнями, потому что они представляют наиболее чувствительные позиции. Небольшое движение цены может спровоцировать крупные каскадные ликвидации на этих уровнях.
</Tip>

Уровни низкого плеча сложнее достичь, но они представляют более крупные позиции. Когда цена достигает кластера ликвидаций низкого плеча, результирующие принудительные ордера существенны.

### Градация по объёму

Когда `volumeGradation` включена, линии с большим предполагаемым объёмом ликвидаций выглядят толще или менее прозрачными. Это помогает мгновенно увидеть, какие уровни имеют больше «топлива» для каскадных ликвидаций.

### Очистка уровней

После того как цена касается уровня ликвидации и запускает принудительные закрытия, этот уровень фактически потреблён. Следите за уровнями, которые ещё не были протестированы — они представляют нетронутую ликвидность.

## Алерты

Уровни ликвидаций поддерживают события алертов `level_touched`. Когда цена достигает уровня ликвидации, срабатывает алерт с деталями о тире плеча и предполагаемом объёме на этом уровне.

| Событие алерта | Условие |
|----------------|---------|
| `level_touched` | Цена достигает предполагаемого уровня ликвидации |

<Note>
Настройте алерты в панели настроек индикатора. Алерты срабатывают один раз при касании уровня и сбрасываются после отхода цены.
</Note>

## Практические примеры

- **Цель ликвидности**: Кластер линий ликвидации шортов с высоким плечом чуть выше текущей цены — потенциальное топливо для шорт-сквиза.
- **Размещение стопов**: Избегайте размещения стопов вблизи плотных уровней ликвидаций — за ними часто охотятся.
- **Ожидание каскада**: Цена приближается к стопке однонаправленных уровней ликвидаций — ожидайте ускорения при пробое первого уровня.
- **Анализ тиров**: Рядом только уровни высокого плеча — движение может быть резким, но кратковременным. Присутствуют уровни низкого плеча — ожидается более устойчивый принудительный поток.

<Warning>
Уровни ликвидаций — это оценки, основанные на распространённых распределениях плеча. Фактические цены ликвидации зависят от индивидуальных входов в позицию и типов маржи (кросс vs. изолированная). Используйте как вероятностный ориентир.
</Warning>
