## Обзор

**Дельта объёма** измеряет разницу между объёмом покупок и объёмом продаж для каждой отдельной свечи. В отличие от CVD, которая накапливается со временем, Дельта объёма обнуляется каждый бар, давая чистое представление о том, кто доминировал в конкретном периоде.

- **Положительная дельта** = больше агрессивных покупок (объём исполнен по цене аск).
- **Отрицательная дельта** = больше агрессивных продаж (объём исполнен по цене бид).

## Как это работает

Для каждого бара:

```
Delta[n] = buyVolume[n] - sellVolume[n]
```

Результат отображается в виде гистограммы на подграфике. Высота каждого столбца представляет величину дисбаланса, а цвет указывает на направление.

## Тип графика

| Свойство | Значение |
|----------|----------|
| Тип | Подграфик |
| Положение | Под основным графиком |
| Источник данных | Классификация покупок/продаж на уровне сделок |

## Настройки

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| positiveColor | Цвет столбцов, где доминируют покупатели | `#22c55e` |
| negativeColor | Цвет столбцов, где доминируют продавцы | `#ef4444` |
| displayMode | Стиль отображения | `columns` |
| highlightAnomalies | Отмечать статистические выбросы | `false` |
| anomalyThreshold | Множитель стандартного отклонения | `2.5` |
| anomalyPeriod | Период ретроспективного анализа для статистики аномалий | `30` |
| gradientIntensity | Интенсивность градиентной заливки (0-1) | `0` |

### Режимы отображения

| Режим | Описание |
|-------|----------|
| `columns` | Столбцы гистограммы (по умолчанию). Лучше всего подходят для быстрого визуального сканирования. |
| `line` | Непрерывная линия, соединяющая значения дельты. |
| `candles` | Представление дельты в стиле OHLC. |

## Интерпретация

### Анализ всплесков

Крупные всплески дельты указывают на внезапный дисбаланс между покупателями и продавцами. Они часто появляются на:

- **Уровнях поддержки/сопротивления** — поглощение или пробой
- **Новостных событиях** — быстрое позиционирование
- **Точках истощения** — финальный толчок перед разворотом

<Tip>
Крупные всплески дельты часто отмечают точки истощения, особенно на уровнях поддержки и сопротивления. Огромный положительный столбец дельты на сопротивлении, который не смог продвинуть цену выше, указывает на то, что покупатели поглощаются.
</Tip>

### Дельта vs. движение цены

| Дельта | Цена | Интерпретация |
|--------|------|---------------|
| Большая положительная | Цена растёт | Подтверждённое давление покупателей |
| Большая положительная | Цена стоит/падает | Покупки поглощаются пассивными продавцами |
| Большая отрицательная | Цена падает | Подтверждённое давление продавцов |
| Большая отрицательная | Цена стоит/растёт | Продажи поглощаются пассивными покупателями |

### Обнаружение аномалий

При включении подсветка аномалий отмечает бары, где дельта превышает скользящее среднее более чем на `anomalyThreshold` стандартных отклонений. Эти бары отображаются с особым стилем для привлечения внимания.

<Note>
Дельта объёма показывает побарную картину. Для кумулятивного тренда используйте CVD. Эти два индикатора дополняют друг друга — Дельта для тайминга, CVD для тренда.
</Note>

## Практические примеры

- **Подтверждение пробоя**: Цена пробивает сопротивление с крупным положительным столбцом дельты — реальный спрос стоит за движением.
- **Поглощение**: Многократные крупные отрицательные столбцы дельты на поддержке, но цена держится — пассивные покупатели поглощают продажи.
- **Кульминационный объём**: Экстремальный всплеск дельты после продолжительного тренда часто сигнализирует об истощении и потенциальном развороте.
