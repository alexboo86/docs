## Обзор

Anchored OI показывает кумулятивное изменение открытого интереса от настраиваемого начала сессии. Вместо просмотра сырых значений OI (которые могут быть очень большими и трудными для интерпретации), вы видите, насколько OI **изменился с открытия сессии**.

- **Положительные значения** — чистое открытие позиций с начала сессии
- **Отрицательные значения** — чистое закрытие позиций с начала сессии

## Как это работает

Открытый интерес представляет общее количество открытых фьючерсных контрактов. Anchored OI отслеживает **дельту от начала сессии**:

```
Anchored OI = Current OI - OI at Session Start
```

Он обнуляется на каждой границе сессии (D1, 4H или 1H), давая чистое представление о потоке позиций внутри каждой сессии.

Ключевой инсайт получается при сочетании направления Anchored OI с направлением цены:

| Anchored OI | Price | Интерпретация |
|-------------|-------|---------------|
| Растёт | Растёт | Новые лонги открываются — бычья убеждённость |
| Растёт | Падает | Новые шорты открываются — медвежья убеждённость |
| Падает | Растёт | Ралли на закрытии шортов — потенциально слабое |
| Падает | Падает | Ликвидация лонгов — потенциально истощающая |

<Tip>
Сравнивайте Anchored OI с Anchored CVD — если оба растут, новые лонги входят с давлением покупок. Если OI растёт, а CVD падает, новые шорты открываются несмотря на покупательскую активность. Эта комбинация — один из самых мощных доступных сигналов позиционирования.
</Tip>

## Интерпретация

### Растущий Anchored OI

Позиции **открываются**. Новые деньги входят в рынок. Это добавляет топливо текущему движению:

- При растущей цене: новые лонги уверены, тренд вероятно продолжится
- При падающей цене: новые шорты агрессивны, нисходящий тренд имеет убеждённость

### Падающий Anchored OI

Позиции **закрываются**. Деньги уходят с рынка. Текущее движение может выдыхаться:

- При растущей цене: закрытие шортов, не реальные покупки — потенциал для разворота после завершения закрытия
- При падающей цене: ликвидация лонгов — после завершения ликвидации давление продавцов может ослабнуть

### Плоский Anchored OI

Нет значительных изменений позиций. Рынок в балансе — существующие участники удерживают позиции, но новых денег не поступает.

## Настройки

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `positiveColor` | Цвет для положительного изменения OI (открытие позиций) | `#3b82f6` |
| `negativeColor` | Цвет для отрицательного изменения OI (закрытие позиций) | `#f97316` |
| `anchor` | Интервал обнуления: `D1`, `4H` или `1H` | `D1` |
| `sessionStartHour` | Час (UTC) начала дневной сессии | `0` |
| `displayMode` | Стиль визуализации: `line`, `columns` или `candles` | `columns` |
| `highlightAnomalies` | Выделение экстремальных изменений OI | `true` |
| `anomalyThreshold` | Число стандартных отклонений для обнаружения аномалий | `2.5` |
| `anomalyPeriod` | Период расчёта аномалий | `50` |
| `gradientIntensity` | Масштабирование прозрачности столбцов по величине | `true` |

## Режимы привязки

### D1 — Дневное обнуление

Обнуляется в `sessionStartHour` (UTC) каждый день. Показывает общее изменение позиций за торговый день. Наиболее полезен для свинг- и дей-трейдинга.

### 4H — Четырёхчасовое обнуление

Обнуляется каждые 4 часа от полуночи UTC. Раскрывает поток позиций в периоде формирования 4H-свечи. Полезен для среднесрочного анализа.

### 1H — Часовое обнуление

Обнуляется каждый час. Показывает микро-уровневые изменения позиций. Лучше всего подходит для скальпинга и очень краткосрочной торговли.

## Режимы отображения

- **Columns** — гистограмма по умолчанию. Синие выше нуля (открытие позиций), оранжевые ниже (закрытие позиций).
- **Line** — непрерывная линия изменения OI от начала сессии.
- **Candles** — OHLC-отображение значений изменения OI за бар.

<Note>
Anchored OI — это индикатор только для фьючерсов. Он требует данных открытого интереса от биржи и недоступен для спотовых рынков.
</Note>

## Обнаружение аномалий

При включении `highlightAnomalies` экстремальные изменения OI внутри сессии выделяются. Обычно они указывают на:

- Вход или выход крупного фонда из позиции
- Каскады маржин-коллов, вынуждающих закрытие позиций
- Хеджирующие потоки, связанные с экспирацией опционов
- Дельта-хеджирование маркет-мейкеров

## Комбинированный анализ: Anchored OI + Anchored CVD

Наиболее мощное использование Anchored OI — в комбинации с Anchored CVD:

| Anchored OI | Anchored CVD | Сигнал |
|-------------|-------------|--------|
| Растёт | Растёт | Новые лонги — сильнейший бычий сигнал |
| Растёт | Падает | Новые шорты — сильнейший медвежий сигнал |
| Падает | Растёт | Закрытие шортов — слабый бычий |
| Падает | Падает | Ликвидация лонгов — слабый медвежий |

## Практические примеры

**Настоящий пробой**: цена пробивает сопротивление. Anchored OI резко растёт — новые позиции открываются на уровне пробоя, подтверждая реальный интерес, а не просто снос стопов.

**Обнаружение шорт-сквиза**: цена резко растёт. Anchored OI падает, а Anchored CVD растёт — шорты принудительно закрываются. После завершения закрытия шортов ралли может остановиться.

**Обнаружение распределения**: цена обновляет сессионные максимумы, но Anchored OI плоский или падает — новые лонги не добавляются на более высоких ценах. Умные деньги могут распределять позиции поздним покупателям.
