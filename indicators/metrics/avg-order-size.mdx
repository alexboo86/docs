## Обзор

Average Order Size рассчитывает средний объём на сделку для каждого бара:

```
Average Order Size = Total Volume / Number of Trades
```

Рост среднего размера ордера указывает на вход крупных игроков на рынок. Снижение среднего размера ордера указывает на доминирование розничной активности с мелкими ордерами.

## Как это работает

Каждая сделка на бирже имеет размер. Вычисляя среднее по всем сделкам в баре, этот индикатор раскрывает **характер** текущей рыночной активности:

- **Большое среднее** — институциональная или «китовая» активность. Меньше, но крупнее ордера.
- **Малое среднее** — розничная активность. Множество мелких ордеров.
- **Всплеск от низкого к высокому** — внезапный вход крупного участника.
- **Постепенное снижение** — институциональный интерес угасает, розница берёт верх.

Это один из наиболее прямых способов определить, когда «умные деньги» входят или выходят с рынка.

<Tip>
Внезапное увеличение среднего размера ордера на ключевом уровне поддержки или сопротивления часто сигнализирует об институциональном интересе. Такие уровни с большей вероятностью удержатся, когда подкреплены крупными ордерами.
</Tip>

## Интерпретация

| Паттерн | Значение |
|---------|----------|
| Растущее среднее + растущая цена | Институциональные покупки — сильный тренд |
| Растущее среднее + падающая цена | Институциональные продажи — сильный нисходящий тренд |
| Падающее среднее + растущая цена | Розничное FOMO — ослабевающее ралли |
| Падающее среднее + падающая цена | Розничная паника — капитуляция может быть близко |
| Всплеск на поддержке | Институциональный бид — уровень вероятно удержится |
| Всплеск на сопротивлении | Институциональный оффер — уровень вероятно удержится |

## Настройки

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `color` | Основной цвет индикатора | `#eab308` |
| `displayMode` | Стиль визуализации: `line`, `columns` или `candles` | `columns` |
| `highlightAnomalies` | Выделение необычно крупных средних ордеров | `true` |
| `anomalyThreshold` | Число стандартных отклонений для обнаружения аномалий | `2.5` |
| `anomalyPeriod` | Период расчёта аномалий | `30` |
| `gradientIntensity` | Масштабирование прозрачности столбцов по величине | `true` |

## Режимы отображения

- **Columns** — гистограмма по умолчанию жёлтого цвета. Высота столбца представляет средний размер ордера.
- **Line** — непрерывная линия. Эффективна для обнаружения трендов в размере ордера со временем.
- **Candles** — OHLC-отображение, показывающее диапазон средних размеров внутри каждого бара.

## Обнаружение аномалий

При включении `highlightAnomalies` бары, где средний размер ордера превышает `anomalyThreshold` стандартных отклонений выше скользящего среднего, выделяются. Эти события указывают на:

- Крупные блочные сделки, попадающие на рынок
- Институциональные алгоритмы, исполняющие крупные ордера
- Корректировки инвентаря маркет-мейкеров
- Хеджирующая активность OTC-десков

## Градиент интенсивности

При включении `gradientIntensity` столбцы с размером ордера ниже среднего выглядят бледными, а выше среднего — при полной непрозрачности. Это естественно привлекает внимание к институциональной активности.

<Note>
Average Order Size значительно различается между активами. BTCUSDT будет иметь совершенно другие базовые значения, чем альткоин с малой капитализацией. Обнаружение аномалий нормализует это, но абсолютные значения следует сравнивать только в рамках одного символа.
</Note>

## Практические примеры

**Фаза накопления**: цена торгуется в узком боковике. Average Order Size постепенно растёт на протяжении нескольких часов — институциональный участник накапливает позицию, не привлекая внимания рынка.

**Розничный блоу-офф топ**: цена резко ускоряется вверх. Average Order Size обваливается до очень низких значений — движение полностью обусловлено мелкими розничными ордерами. Это классический сигнал кульминационной вершины.

**Институциональная защита уровня**: цена падает к ключевому уровню поддержки. Average Order Size резко взлетает на отскоке — крупный покупатель вмешался. Поддержка более достоверна.

**Угасание институционального интереса**: после сильного трендового движения Average Order Size снижается бар за баром. Крупные игроки, инициировавшие движение, отходят в сторону — тренд может исчерпываться.
