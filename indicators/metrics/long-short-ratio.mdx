## Обзор

Long/Short Ratio отображает соотношение длинных к коротким позициям по данным биржи Binance. Доступен для трёх категорий трейдеров:

- **Global** — все трейдеры на бирже
- **Top Position** — топ-трейдеры по размеру общей позиции
- **Top Account** — топ-трейдеры по стоимости аккаунта

Когда соотношение выше 1.0, лонгов больше, чем шортов. Ниже 1.0 — шорты доминируют.

## Как это работает

Binance публикует агрегированные данные соотношения лонгов/шортов через регулярные интервалы для фьючерсных рынков. Этот индикатор получает и отображает эти данные напрямую, сегментированные по категориям трейдеров.

Три категории раскрывают различные аспекты рыночного позиционирования:

- **Global ratio** отражает позиционирование всего рынка. Экстремальные значения часто служат контртрендовыми сигналами — когда все в лонге, рынок может быть перекредитован для лонг-сквиза.
- **Top Position ratio** показывает, как позиционированы держатели крупнейших позиций. Это часто более информированные участники.
- **Top Account ratio** показывает позиционирование по стоимости аккаунта. Трейдеры с высоким капиталом могут иметь иной взгляд, чем толпа.

<Tip>
Когда все три соотношения расходятся с направлением цены, это создаёт сильный контртрендовый сигнал. Например, если все три показывают экстремальное длинное позиционирование, а цена останавливается, лонг-сквиз становится вероятным.
</Tip>

## Интерпретация

| Паттерн | Значение |
|---------|----------|
| Все соотношения > 1.5 | Рынок сильно в лонг — риск сквиза |
| Все соотношения < 0.7 | Рынок сильно в шорт — риск шорт-сквиза |
| Global лонг + Top шорт | Розница бычья, умные деньги медвежьи — медвежий сигнал |
| Global шорт + Top лонг | Розница медвежья, умные деньги бычьи — бычий сигнал |
| Все соотношения сходятся к 1.0 | Сбалансированное позиционирование — нет направленного преимущества |

### Контртрендовые сигналы

Long/Short Ratio наиболее полезен как **контртрендовый индикатор**. Экстремально перегруженные позиции склонны к резкому разворачиванию:

- Экстремальные лонги + цена у сопротивления = высоковероятная установка для шорта
- Экстремальные шорты + цена у поддержки = высоковероятная установка для лонга

### Дивергенция умных денег

Когда Global ratio расходится с Top Position и Top Account ratio, это раскрывает разрыв между розничным и институциональным позиционированием. Следуйте за «топовыми» категориями для направленного перевеса.

## Настройки

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `period` | Период данных: `5m`, `15m`, `30m`, `1h`, `2h`, `4h`, `6h`, `12h`, `1d` | `5m` |
| `showGlobal` | Показать глобальную линию L/S ratio | `true` |
| `showTopPosition` | Показать линию L/S ratio топ-позиций | `true` |
| `showTopAccount` | Показать линию L/S ratio топ-аккаунтов | `true` |
| `globalColor` | Цвет линии глобального соотношения | `#3b82f6` |
| `topPositionColor` | Цвет линии топ-позиций | `#f59e0b` |
| `topAccountColor` | Цвет линии топ-аккаунтов | `#a855f7` |
| `lineWidth` | Толщина линий | `1.5` |
| `displayMode` | Стиль визуализации: `line` или `columns` | `line` |

## Период данных

Настройка `period` контролирует гранулярность данных L/S ratio от Binance:

| Period | Применение |
|--------|------------|
| `5m` | Скальпинг — быстрые изменения позиционирования |
| `15m`–`30m` | Внутридневная торговля |
| `1h`–`4h` | Свинг-трейдинг |
| `12h`–`1d` | Позиционная торговля — более широкий сентимент |

<Note>
Данные Long/Short Ratio поступают напрямую от Binance. Они доступны только для символов Binance Futures. Данные могут иметь небольшую задержку (обычно 1-2 минуты) по сравнению с данными цены в реальном времени.
</Note>

## Режимы отображения

- **Line** — по умолчанию. Три отдельные цветные линии относительно базовой линии 1.0. Лучше всего для сравнения трёх категорий.
- **Columns** — гистограмма. Менее распространён, но полезен при фокусе на одной категории.

## Практические примеры

**Установка на лонг-сквиз**: все три L/S ratio выше 2.0 (экстремальное длинное позиционирование). Цена приближается к крупному уровню сопротивления и останавливается. Небольшая распродажа вызывает ликвидации, и перегруженная сторона лонгов быстро разворачивается — цена резко падает.

**Фейд умных денег**: Global ratio показывает 1.8 (розница сильно в лонг). Top Position ratio на 0.8 (институционалы в шорт). Эта дивергенция указывает, что «умные деньги» ожидают снижения. Торгуйте против розничной толпы.

**Подтверждающий сигнал**: цена пробивает ключевой уровень. Global L/S ratio начинает расти от 1.0 к 1.3 — участники добавляют лонги после пробоя, подтверждая, что движение имеет участие, а не просто снос стопов.
