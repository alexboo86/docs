## Обзор

Enhanced Liquidations — это расширенная версия базового индикатора Liquidations. Он добавляет **фильтрацию по корзинам размера позиции**, позволяя сфокусироваться на ликвидациях, которые действительно двигают рынок.

- **Ликвидации лонгов** — принудительное закрытие длинных позиций (показаны одним цветом)
- **Ликвидации шортов** — принудительное закрытие коротких позиций (показаны другим цветом)
- **Фильтр по размеру** — показ только ликвидаций выше настраиваемого порога размера позиции

## Как это работает

Когда маржа трейдера падает ниже уровня поддержания, биржа принудительно закрывает его позицию — это ликвидация. Enhanced Liquidations фиксирует эти события и категоризирует их по:

1. **Направлению** — была ликвидирована длинная или короткая позиция?
2. **Размеру** — насколько крупной была ликвидированная позиция?

Система корзин по размеру позволяет отфильтровать шум мелких ликвидаций и сфокусироваться на значительных принудительных закрытиях, создающих каскадное влияние на цену.

### Корзины размера

| Bucket | Минимальный размер позиции | Применение |
|--------|---------------------------|------------|
| `0` | Все ликвидации | Полная картина, включая шум |
| `1` | 1,000 USD+ | Фильтрация микро-позиций |
| `2` | 10,000 USD+ | Средние ликвидации |
| `3` | 100,000 USD+ | Только крупные ликвидации |
| `4` | 1,000,000 USD+ | Только китовые ликвидации |

<Tip>
Фильтруйте до корзины 10K+ (minBucket: 2), чтобы видеть только значительные ликвидации. Крупные каскады ликвидаций часто отмечают локальные ценовые экстремумы — когда большие позиции уничтожаются, принудительное давление продаж/покупок исчерпывается и цена разворачивается.
</Tip>

## Интерпретация

### Ликвидации лонгов (отображаются выше нулевой линии)

Длинные позиции принудительно закрываются. Биржа продаёт их позицию на рынке, добавляя давление продаж. Кластеры ликвидаций лонгов при падении указывают на **каскад лонг-сквиза** — каждая ликвидация толкает цену ниже, вызывая новые ликвидации.

### Ликвидации шортов (отображаются ниже нулевой линии)

Короткие позиции принудительно закрываются. Биржа выкупает их позицию, добавляя давление покупок. Кластеры ликвидаций шортов при росте указывают на **шорт-сквиз** — каждая ликвидация толкает цену выше, вызывая новые ликвидации.

### Обнаружение каскадов

Наиболее мощный сигнал Enhanced Liquidations — **каскад** — самоусиливающийся цикл, где ликвидации вызывают новые ликвидации:

1. Цена движется против перекредитованных позиций
2. Первая волна ликвидаций срабатывает
3. Принудительные ордера толкают цену дальше
4. Вторая волна срабатывает
5. Повторяется до полного вымывания левереджа

Эти каскады часто заканчиваются на экстремумах, которые отмечают значительные точки разворота.

## Настройки

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `longColor` | Цвет ликвидаций лонгов | `#22c55e` |
| `shortColor` | Цвет ликвидаций шортов | `#ef4444` |
| `mode` | Показывать объём или количество ликвидаций | `volume` |
| `minBucket` | Минимальная корзина размера позиции (0–4) | `0` |
| `displayMode` | Стиль визуализации: `line`, `columns` или `candles` | `columns` |
| `highlightAnomalies` | Выделение экстремальных событий ликвидации | `true` |
| `anomalyThreshold` | Число стандартных отклонений для обнаружения аномалий | `2.5` |
| `anomalyPeriod` | Период расчёта аномалий | `30` |
| `gradientIntensity` | Масштабирование прозрачности столбцов по величине | `true` |
| `minOpacity` | Минимальная прозрачность для градиентного масштабирования | `0.3` |
| `barWidth` | Коэффициент ширины столбца (0.0–1.0) | `0.6` |

## Режим: Volume vs Count

- **Volume** (по умолчанию) — показывает общую долларовую стоимость ликвидированных позиций. Лучше для оценки рыночного влияния, поскольку одна крупная ликвидация важнее множества мелких.
- **Count** — показывает количество событий ликвидации. Лучше для оценки того, сколько трейдеров затронуто, независимо от размера позиции.

## Режимы отображения

- **Columns** — гистограмма по умолчанию. Ликвидации лонгов одним цветом, шортов другим. Высота показывает величину.
- **Line** — непрерывная линия активности ликвидаций. Полезна для обнаружения трендов в частоте ликвидаций.
- **Candles** — OHLC-отображение значений ликвидаций за бар.

## Обнаружение аномалий

При включении `highlightAnomalies` бары ликвидаций, превышающие `anomalyThreshold` стандартных отклонений от скользящего среднего, визуально отмечаются. Аномалии ликвидаций — одни из самых значимых событий на фьючерсных рынках:

- Крупные каскадные события, вымывающие левереджные позиции
- Биржевые волны ликвидаций во время флеш-крашей
- Единичные китовые ликвидации, двигающие рынок

## Градиент интенсивности

При включении `gradientIntensity` столбцы окрашиваются по величине ликвидации. Мелкие ликвидации выглядят бледными (до `minOpacity`), а крупные каскады — при полной непрозрачности. Это естественно выделяет события, которые имеют наибольшее значение.

<Note>
Enhanced Liquidations — это индикатор только для фьючерсов. Данные ликвидаций получены из отчётов биржи о событиях ликвидации. Некоторые биржи могут иметь небольшую задержку в отчётности, поэтому бары ликвидаций могут появляться с задержкой 1-2 секунды после соответствующего движения цены.
</Note>

## Практические примеры

**Каскадный разворот**: цена резко падает. Ликвидации лонгов взлетают до уровня аномалий в корзине 10K+. По мере исчерпания каскада объём ликвидаций снижается — это часто точка максимальной боли и потенциальная зона разворота.

**Вымывание левереджа**: после периода роста открытого интереса и спокойного рынка, внезапное движение вызывает волну ликвидаций. Резкий всплеск Enhanced Liquidations подтверждает, что перекредитованные позиции вымыты — рынок здоровее для следующего движения.

**Подтверждение сквиза**: цена пробивает сопротивление. Ликвидации шортов взлетают в корзине 100K+ — крупные шорты сжимаются. Принудительные покупки от этих ликвидаций добавляют топливо пробою.

**Анализ корзин**: сравните обзор всех корзин (minBucket: 0) с обзором 10K+ (minBucket: 2). Если мелкие ликвидации высоки, а крупных нет, движение не имеет достаточной силы для запуска каскада — вероятно незначительная встряска.
