## Обзор

**Открытый интерес (OI)** представляет общее количество открытых фьючерсных контрактов, которые не были закрыты. Это мера участия рынка и вовлечённости капитала — не объём, а количество позиций, открытых в данный момент.

- **Растущий OI** = новые деньги входят на рынок (открываются новые позиции).
- **Падающий OI** = позиции закрываются (трейдеры выходят).

OI уникален для рынков деривативов и предоставляет информацию, которую объём сам по себе дать не может: поддерживается ли текущее движение цены новым капиталом.

## Как это работает

Открытый интерес увеличивается, когда новый покупатель и новый продавец оба открывают новые позиции. Он уменьшается, когда существующий лонг и существующий шорт закрывают свои позиции друг против друга. OI остаётся неизменным, когда существующая позиция передаётся от одного участника другому.

| Покупатель | Продавец | Изменение OI |
|------------|----------|--------------|
| Новый лонг | Новый шорт | +1 |
| Закрытие лонга | Закрытие шорта | -1 |
| Новый лонг | Закрытие лонга (передача) | 0 |

## Настройки

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| lineColor | Цвет линии | `#38bdf8` |
| lineWidth | Толщина линии | `1.5` |
| displayMode | Стиль отображения | `line` |
| highlightAnomalies | Отмечать статистические выбросы | `false` |
| anomalyThreshold | Множитель стандартного отклонения | `2.5` |
| anomalyPeriod | Период ретроспективного анализа для статистики аномалий | `50` |
| gradientIntensity | Интенсивность градиентной заливки (0-1) | `0` |

### Режимы отображения

| Режим | Описание |
|-------|----------|
| `line` | Непрерывная линия (по умолчанию). Лучше всего подходит для отслеживания тренда OI. |
| `columns` | Столбчатая диаграмма, показывающая значение OI за период. |
| `candles` | OHLC изменений OI — полезно для обнаружения разворотов OI. |

## Интерпретация

### Матрица OI + Цена

Классическая система анализа OI совместно с ценой:

| Цена | OI | Интерпретация |
|------|-----|---------------|
| Растёт | Растёт | Сильный тренд — новые деньги поддерживают ралли |
| Растёт | Падает | Шорт-сквиз — шорты закрываются, новых покупок нет |
| Падает | Растёт | Сильный нисходящий тренд — новые шорты входят |
| Падает | Падает | Капитуляция лонгов — лонги закрываются, давление продаж ослабевает |

<Tip>
Растущая цена с растущим OI сигнализирует о сильном тренде, подкреплённом новым капиталом. Растущая цена с падающим OI указывает на шорт-сквиз — ралли может не иметь устойчивости.
</Tip>

### Всплески OI

Внезапные крупные увеличения OI указывают на массовое новое позиционирование. Они часто появляются перед крупными движениями, когда участники вкладывают капитал в ожидании волатильности (например, перед новостными событиями, на ключевых технических уровнях).

### Снижение OI

Устойчивое снижение OI во время тренда говорит о том, что тренд теряет участников и может приближаться к истощению. Трейдеры фиксируют прибыль, а не наращивают позиции.

<Warning>
Данные OI специфичны для биржи. Cluster Terminal показывает OI для выбранной биржи. Суммарный OI рынка по всем биржам может отличаться.
</Warning>

## Обнаружение аномалий

Когда `highlightAnomalies` включён, бары, где изменение OI превышает скользящее среднее более чем на настроенный порог, отмечаются. Крупные аномалии OI часто предшествуют значительным движениям цены.

## Практические примеры

- **Проверка силы тренда**: Восходящий тренд с устойчиво растущим OI — движение имеет убеждённость, удерживайте позицию.
- **Обнаружение сквиза**: Цена резко растёт, но OI падает — вероятен шорт-сквиз, будьте осторожны с лонгами.
- **Позиционирование перед событием**: OI резко растёт перед запланированным событием — трейдеры делают ставки на исход.
- **Капитуляция**: Быстрое снижение OI после падения — принудительные закрытия завершаются, потенциальное дно.
