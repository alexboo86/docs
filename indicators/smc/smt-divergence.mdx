SMT (Smart Money Technique) Divergence сравнивает ценовую структуру двух коррелированных активов (например, BTC и ETH). Когда один актив обновляет экстремум, а второй не подтверждает движение — это сигнал потенциального разворота.

## Принцип работы

Два коррелированных актива обычно двигаются синхронно. Когда синхронность нарушается на ключевых swing-точках — это указывает на манипуляцию или перераспределение позиций крупными участниками.

**Примеры:**
- BTC делает Higher High, а ETH — Lower High → медвежья дивергенция
- BTC делает Lower Low, а ETH — Higher Low → бычья дивергенция

## Режимы

### Fractal (фрактальный)

Сравнивает Williams-фракталы (pivot highs/lows) обоих активов.

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| **Размер фрактала** | 3 или 5 свечей для определения пивота | 5 |
| **Только соседние** | Показывать только последний сигнал | Нет |
| **Макс. расстояние** | Допустимое смещение между фракталами (в барах) | 20 |
| **Мин. расхождение %** | Минимальная разница для фильтрации шума | 0.1% |

Фракталы матчатся по времени (ближайший в окне ±N баров), а не по индексу — это учитывает, что пивоты на разных активах формируются с небольшим смещением.

### Level (по уровням)

Детектирует sweep session levels: когда один актив пробивает уровень предыдущего периода, а второй — нет.

Отслеживаемые уровни:
- **PDH / PDL** — максимум / минимум предыдущего дня
- **PWH / PWL** — максимум / минимум предыдущей недели
- **PMH / PML** — максимум / минимум предыдущего месяца

Sweep high → медвежий сигнал (ликвидити-граб). Sweep low → бычий сигнал.

### Both (оба)

Комбинирует оба режима на одном графике.

## Выбор референсного символа

Нажмите на кнопку с лупой рядом с названием символа, чтобы открыть стандартный диалог поиска. Выбранный символ станет референсным для сравнения.

Рекомендуемые пары:
- **BTCUSDT → ETHUSDT** — основная крипто-пара
- **BTCUSDT → SOLUSDT** — альтернативная пара
- **ETHUSDT → SOLUSDT** — alt-to-alt

## Визуализация

| Элемент | Описание |
|---------|----------|
| **Точки** | Маркеры на графике в точках дивергенции (зелёные — бычьи, красные — медвежьи) |
| **Линии** | Соединяют предыдущий и текущий пивот, показывая направление движения |
| **Линия реф. цены** | Нормализованная цена второго актива поверх графика (опционально) |

## Настройки оформления

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| **Цвет бычьей** | Цвет бычьих сигналов | #22c55e |
| **Цвет медвежьей** | Цвет медвежьих сигналов | #ef4444 |
| **Размер точки** | Размер маркера дивергенции | 8 |
| **Толщина линии** | Толщина соединительных линий | 2 |
| **Стиль линии** | Solid / Dash / Dot | Dash |
| **Показать метки** | Текстовые лейблы (HH vs LH и т.д.) | Да |
| **Линия реф. цены** | Overlay цены второго актива | Нет |

## Алерты

Поддерживаются стандартные режимы алертов. При срабатывании уведомление содержит описание дивергенции (например, "BTC HH vs ETH LH" или "PDH sweep").
