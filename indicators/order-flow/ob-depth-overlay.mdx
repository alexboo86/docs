# OB Depth Overlay

OB Depth Overlay рисует **цветные полосы за каждой свечой**, отображающие доминирование бидов/асков в ордербуке на разных уровнях глубины. В отличие от OB Imbalance (который рисуется в подграфике), этот оверлей накладывается прямо на ценовой график, давая мгновенную визуальную оценку того, где концентрируется давление покупателей или продавцов.

## Как это работает

На закрытии каждой свечи индикатор берёт последний снимок ордербука и вычисляет доминирование на **5 настраиваемых уровнях глубины**, измеряемых в процентах от последней цены:

```
Доминирование = (Bids − Asks) / (Bids + Asks) × 100
```

| Уровень | Диапазон по умолчанию | Значение |
|---------|----------------------|----------|
| Уровень 1 | 0–1% | Ближайший к цене — узкие спреды |
| Уровень 2 | 1–2.5% | Близкая глубина |
| Уровень 3 | 2.5–5% | Средняя глубина |
| Уровень 4 | 5–10% | Расширенная глубина (выключен по умолчанию) |
| Уровень 5 | 10–25% | Полная глубина книги (выключен по умолчанию) |

- **Положительное доминирование** → зелёная полоса (биды преобладают) под свечой
- **Отрицательное доминирование** → красная полоса (аски преобладают) над свечой
- **Прозрачность полосы** масштабируется силой доминирования (сильнее = более непрозрачно)

## Настройки

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| **Максимальная прозрачность** | 85% | Максимальная непрозрачность полосы при 100% доминировании |
| **Мин. порог бидов** | 10 | Минимальное доминирование бидов (%) для показа зелёной полосы |
| **Мин. порог асков** | 10 | Минимальное доминирование асков (%) для показа красной полосы |
| **Переключатели уровней** | Уровни 1-3 вкл, 4-5 выкл | Включение/выключение отдельных уровней |
| **Цвета уровней** | Зелёный (биды), Красный (аски) | Настройка цветов для каждого уровня |

## Применение в трейдинге

- **Конфлюенция с прайс-экшен**: зелёные полосы под уровнями поддержки подтверждают пассивный покупательский интерес
- **Обнаружение дивергенций**: цена растёт, но красные полосы увеличиваются — сигнал потенциального разворота
- **Многоуровневая глубина**: сравнивайте узкие уровни (0-1%) с широкими (2.5-5%) — узкое доминирование часто опережает цену, широкое даёт контекст
- **Комбинируйте с Delta**: оверлей показывает лимитные ордера, дельта — агрессивное исполнение

## Примечания

- Данные берутся из снимков ордербука каждые ~15 секунд
- Каждая свеча показывает **последний снимок** интервала свечи (candle-close printing)
- Доступно только на биржах с данными ордербука (Binance Futures и Spot)
