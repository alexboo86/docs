# Diamond Alerts

Diamond Alerts обнаруживают потенциальные точки разворота рынка, находя **конфлюенцию между ордербуками Spot и Perpetual**. Когда оба рынка одновременно показывают сильное доминирование бидов или асков на одном уровне глубины, формируется Diamond-сигнал — указывающий на согласованное институциональное позиционирование на обеих площадках.

{/* screenshot: Diamond-маркеры на графике — зелёные ромбы под свечами, красные над */}

## Как это работает

Для каждой свечи система:

1. Берёт последний снимок ордербука с **Perpetual** (фьючерсного) рынка
2. Берёт последний снимок ордербука со **Spot** рынка
3. Вычисляет доминирование bid/ask на настроенном уровне глубины (по умолчанию: 1-2.5% от цены)
4. Если оба рынка превышают свои пороги → **Diamond-сигнал**

### Типы сигналов

| Сигнал | Условие | Маркер |
|--------|---------|--------|
| **Diamond Long** 💎 | Perp биды > 40% И Spot биды > 70% | Зелёный ◆ под свечой |
| **Diamond Short** 💎 | Perp аски > 40% И Spot аски > 70% | Красный ◆ над свечой |
| **Double Diamond Long** 💎💎 | Оба уровня (primary И secondary) показывают Long конфлюенцию | Увеличенный зелёный ◆ с контуром |
| **Double Diamond Short** 💎💎 | Оба уровня (primary И secondary) показывают Short конфлюенцию | Увеличенный красный ◆ с контуром |

### Пороги по умолчанию

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| Порог бидов Perp | 40% | Минимальное доминирование бидов на фьючерсах |
| Порог бидов Spot | 70% | Минимальное доминирование бидов на споте |
| Порог асков Perp | 40% | Минимальное доминирование асков на фьючерсах |
| Порог асков Spot | 70% | Минимальное доминирование асков на споте |
| Основной диапазон | 1–2.5% | Главный уровень глубины для детекции |
| Вторичный диапазон | 2.5–5% | Расширенный уровень для Double Diamond |

Пороги асимметричны по замыслу: на спотовом рынке требуется более сильное доминирование, потому что он отражает реальное намерение покупки/продажи без кредитного плеча.

## Настройки

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| **Одинарный Diamond** | Вкл | Включить детекцию одинарных diamond |
| **Двойной Diamond** | Вкл | Включить детекцию двойных diamond |
| **Пороги Perp/Spot** | См. выше | Настраиваемые пороги по направлению |
| **Основной/Вторичный диапазон** | 1-2.5% / 2.5-5% | Уровни глубины для анализа |
| **Цвет Long** | Зелёный | Цвет маркера для лонгов |
| **Цвет Short** | Красный | Цвет маркера для шортов |
| **Размер маркера** | 14 | Размер ромба в пикселях |
| **Прозрачность маркера** | 90% | Прозрачность маркеров |
| **Правила алертов** | Все включены (trumpet) | Настраиваемые звуковые алерты по типу сигнала |

## Применение в трейдинге

- **Входы на развороте**: Diamond Long на поддержке = сильный сигнал на покупку с подтверждением обоих рынков
- **Фильтр конфлюенции**: используйте Diamond-сигналы для фильтрации входов от других индикаторов (RSI Divergence, FVG и т.д.)
- **Double Diamond = высокая уверенность**: когда оба уровня глубины совпадают, сигнал сильнее
- **Фильтр по таймфрейму**: лучше всего работает на 1m-15m, где данные ордербука обновляются часто

## Примечания

- Доступно только на **фьючерсных графиках** (`binancef`) — индикатор автоматически загружает спотовые данные
- Снимки ордербуков Spot и Perpetual должны быть доступны для одной и той же свечи
- Максимум один diamond на свечу (дедупликация предотвращает множественные сигналы)
- Звуковые алерты поддерживают режим повтора — срабатывают на каждый новый diamond
