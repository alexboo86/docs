# Bar Statistics

Bar Statistics отрисовывает компактный **статистический футер под каждой свечой**, разбивая каждый бар на основные компоненты потока ордеров: объём покупок, объём продаж, общий объём, дельта, сделки покупок и сделки продаж. Вместо чтения отдельных ячеек кластера вы получаете мгновенную сводку по бару, раскрывающую баланс участия одним взглядом.

## Как это работает

Для каждой свечи на графике Bar Statistics извлекает данные из базового кластера (футпринта) и отображает их в виде набора подписанных строк под свечой. Каждая строка представляет отдельную метрику, а её ячейка окрашена по интенсивности для отражения величины значения относительно других баров.

Индикатор обрабатывает следующие метрики для каждого бара:

| Метрика | Расчёт |
|---------|--------|
| **Buy Volume** | Сумма всего объёма, исполненного по цене аск (агрессивные покупатели) |
| **Sell Volume** | Сумма всего объёма, исполненного по цене бид (агрессивные продавцы) |
| **Total Volume** | Buy Volume + Sell Volume |
| **Delta** | Buy Volume − Sell Volume |
| **Buy Trades** | Количество индивидуальных сделок покупки |
| **Sell Trades** | Количество индивидуальных сделок продажи |

Каждая метрика независимо окрашивается по интенсивности с использованием выбранного режима градации, позволяя визуально сравнивать относительные величины между барами.

## Тип индикатора

**Side Panel** (футер-таблица) — отрисовывается как строки цветных ячеек непосредственно под свечным графиком. Не занимает отдельную панель подграфика.

## Настройки

| Parameter | Тип | Default | Описание |
|-----------|-----|---------|----------|
| `showBuyVol` | boolean | `true` | Показывать строку объёма покупок |
| `showSellVol` | boolean | `true` | Показывать строку объёма продаж |
| `showTotalVol` | boolean | `true` | Показывать строку общего объёма |
| `showDeltaVol` | boolean | `true` | Показывать строку дельты (покупки − продажи) |
| `showBuyTrades` | boolean | `true` | Показывать строку количества сделок покупок |
| `showSellTrades` | boolean | `true` | Показывать строку количества сделок продаж |
| `grading` | select | `visible` | Режим нормализации интенсивности цвета (см. ниже) |
| `buyColor` | color | `#22c55e` | Цвет для статистики стороны покупок |
| `sellColor` | color | `#ef4444` | Цвет для статистики стороны продаж |

### Переключение строк

Вы можете включить или отключить любую комбинацию из шести строк. Для минимальной настройки покажите только `Delta`, чтобы видеть чистый поток ордеров. Для полного анализа включите все шесть строк для сравнения объёма, дельты и количества сделок бок о бок.

## Режимы градации

Режим градации определяет, как нормализуются цвета ячеек — то есть какой диапазон значений соответствует самой светлой и самой тёмной интенсивности цвета. Это одна из важнейших настроек, поскольку она контролирует, что означает «высокое» и «низкое» визуально.

| Режим | Диапазон нормализации | Лучше для |
|-------|----------------------|-----------|
| `individual` | Каждый бар окрашивается относительно только своих собственных метрик | Анализ внутреннего баланса одного бара |
| `daily` | Значения нормализуются по всем барам текущего торгового дня | Внутридневной сессионный анализ — поиск экстремумов дня |
| `visible` | Значения нормализуются по всем барам, видимым на экране | Сканирование текущего вида для выявления относительных аномалий |
| `whole` | Значения нормализуются по всему загруженному набору данных | Долгосрочное сравнение — определение исторически значимых баров |

### Выбор режима градации

- **`visible`** (по умолчанию) — наиболее универсальный выбор. При прокрутке или масштабировании нормализация цвета пересчитывается, всегда подсвечивая выделяющиеся бары в текущем виде.
- **`daily`** полезен для внутридневных трейдеров, которые хотят видеть, где произошла пиковая активность сессии.
- **`whole`** идеален для старших таймфреймов, где нужно сравнить сегодняшнюю активность с неделями истории.
- **`individual`** имеет узкое применение: показывает, доминировал ли объём покупок или продаж внутри каждого бара, но не сравнивает бары между собой.

<Tip>
Используйте градацию **visible** для быстрого обнаружения аномалий объёма в текущем виде. Самые яркие ячейки сразу привлекают внимание к барам с выдающимся участием — потенциальной институциональной активности или всплесков на новостях.
</Tip>

## Практическое использование

### Определение институционального участия

Когда бар показывает очень высокий общий объём, но относительно маленькую дельту, это указывает на интенсивное двустороннее участие — и покупатели, и продавцы активны. Это часто происходит в зонах институционального накопления или распределения.

### Расхождение количества сделок и объёма

Сравните строки количества сделок со строками объёма:

- **Высокий объём + малое количество сделок** = несколько крупных ордеров (институциональная / китовая активность)
- **Низкий объём + большое количество сделок** = множество мелких ордеров (ритейл / алгоритмический шум)

Это расхождение — один из наиболее надёжных следов активности умных денег.

### Подтверждение дельтой

Используйте строку дельты для подтверждения направления пробоя. Свеча пробоя с большой дельтой в направлении пробоя указывает на подлинную агрессию. Пробой с нейтральной или противоположной дельтой с большей вероятностью окажется ложным.

## Сочетание с другими индикаторами

Bar Statistics естественно сочетается с:

- **Clouds** — когда Clouds подсвечивает определённый ценовой уровень, проверьте Bar Statistics на этом баре для полной разбивки.
- **CVD (Cumulative Volume Delta)** — CVD показывает нарастающий итог, а Bar Statistics — вклад каждого бара.
- **Volume Profile** — профиль показывает, где объём сконцентрирован по цене; Bar Statistics показывает, как объём разбился по направлению на каждом баре.

<Tip>
Bar Statistics — один из самых лёгких индикаторов, которые можно добавить: он читает данные, уже загруженные в кластер, и просто переформатирует их. Нет штрафа производительности за включение всех шести строк.
</Tip>

## Оповещения

Bar Statistics не поддерживает оповещения. Для оповещений на основе объёма используйте индикаторы Delta или Volume с включённой подсветкой аномалий.

## Итог

Bar Statistics даёт вам структурированную, цветокодированную декомпозицию каждой свечи без необходимости чтения сырого футпринта. Он отвечает на ключевые вопросы потока ордеров — кто был более агрессивен, сколько участников было вовлечено и как этот бар сравнивается с соседними — одним взглядом.
