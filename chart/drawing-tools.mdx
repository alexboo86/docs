Cluster Terminal включает более 60 профессиональных инструментов рисования и аннотации для технического анализа, распознавания паттернов, планирования сделок и аннотирования графиков. Все инструменты организованы в 10 категорий, доступных из панели рисования на тулбаре. Каждый рисунок сохраняется между сессиями как часть рабочего пространства и включается в облачную синхронизацию.

## Глобальные настройки рисования

Перед изучением отдельных инструментов настройте глобальные параметры, влияющие на все операции рисования.

### Режим магнита

При включении точки привязки притягиваются к ближайшему OHLC-значению (open, high, low или close) ближайшей свечи. Это устраняет неточное ручное размещение.

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|
| **Magnet** | Включить/выключить привязку к цене | Выкл |
| **Magnet Radius** | Расстояние в пикселях для привязки | 10px |

Переключайте режим магнита клавишей `M` в любой момент. Магнит работает со всеми точечными инструментами рисования — трендовые линии, Фибоначчи, Ганн, фигуры и паттерны.

<Tip>
Включайте режим магнита при рисовании линий поддержки/сопротивления, уровней коррекции Фибоначчи и паттернов. Отключайте при рисовании произвольных аннотаций, текстовых меток и приблизительных зон, где точная привязка к цене не важна.
</Tip>

### Режим непрерывного рисования

При включении выбранный инструмент остаётся активным после завершения рисунка, позволяя размещать несколько экземпляров без повторного выбора инструмента. При выключении курсор возвращается к стандартному указателю после каждого завершённого рисунка.

### Блокировка / скрытие всех объектов

| Действие | Описание |
|----------|----------|
| **Lock Objects** | Предотвращает перемещение и изменение размера всех рисунков. Включается сквозной клик. |
| **Hide Objects** | Делает все рисунки невидимыми без удаления. Полезно для создания чистых скриншотов. |

### Боковая панель рисования (DrawSidebar)

Постоянная панель инструментов рисования, закреплённая у левого края графика. Включается/выключается через настройки рисования. Содержит категории инструментов с выпадающими подменю, не перекрывая основной график. Заменяет плавающий попап для более удобного доступа к инструментам.

### Линейка (Ruler)

Инструмент измерения расстояния между двумя точками графика. Активируется **средним кликом мыши** (колесо) с автоматическим включением магнита, или выбором из панели инструментов.

Информационный блок линейки показывает:

| Строка | Описание |
|--------|----------|
| **Bars** | Количество баров между точками |
| **Time** | Временной интервал |
| **Price** | Разница цен |
| **Ticks** | Разница в тиках |
| **Change%** | Процентное изменение |
| **Volume** | Суммарный объём |
| **Trades** | Количество сделок |
| **Delta** | Дельта (Buy - Sell) |
| **Bid** | Суммарный Bid |
| **Ask** | Суммарный Ask |

Каждая строка может быть включена/выключена индивидуально. Цвет, размер шрифта и стиль линии (solid/dashed/dotted) настраиваются.

### Панель избранного

Добавляйте часто используемые инструменты в панель избранного для доступа в один клик. Чтобы добавить инструмент в избранное:

1. Выберите инструмент из панели рисования.
2. Нажмите **иконку звезды** в панели параметров инструмента.
3. Инструмент появится в полосе избранного под тулбаром.

Чтобы удалить инструмент из избранного, нажмите **кнопку X** на его записи в избранном или нажмите иконку звезды снова при выбранном инструменте.

### Стили по умолчанию

Каждый инструмент рисования запоминает последний применённый стиль. При создании нового рисунка того же типа он наследует последний использованный цвет обводки, цвет заливки, ширину линии и другие свойства. Это обеспечивает визуальную согласованность без ручной перенастройки.

## Категории инструментов

### 1. Курсоры (4 инструмента)

| Инструмент | Описание |
|------------|----------|
| **Cursor** | Стандартный указатель для панорамирования, выделения и взаимодействия с элементами графика. |
| **Dot Cursor** | Перекрестие-точка для точного позиционирования координат. Показывает точную цену и время в позиции указателя. |
| **Arrow Cursor** | Направленная стрелка-указатель. Функциональный эквивалент стандартного курсора с другим визуалом. |
| **Eraser** | Нажмите на любой рисунок для немедленного удаления. Удобен для быстрой очистки без выделения и нажатия Delete. |

### 2. Трендовые линии (6 инструментов)

Базовые инструменты технического анализа для определения направления, уровней и каналов.

| Инструмент | Точки | Описание |
|------------|-------|----------|
| **Trend Line** | 2 | Прямая линия между двумя точками привязки. Фундаментальный инструмент для обозначения направления тренда. Поддерживает продление в обе стороны (настраиваемое). |
| **Ray** | 2 | Линия, начинающаяся от первой точки и бесконечно продлённая в направлении второй. Полезна для проецирования направления тренда в будущее. |
| **Horizontal Line** | 1 | Линия ценового уровня, проходящая через весь видимый график. Поддерживает ценовые оповещения (пересечение, выше, ниже). Наиболее часто используемый инструмент рисования. |
| **Vertical Line** | 1 | Временная метка, проходящая от верха до низа. Полезна для отметки событий, выхода новостей или границ сессий. |
| **Arrow** | 2 | Направленная линия с наконечником-стрелкой. Настройте стиль стрелки (заполненная, открытая, без) и какие концы отображают наконечник. |
| **Parallel Channel** | 3 | Две параллельные линии, определяющие ценовой канал. Поставьте две точки для первой линии, затем третью для задания ширины канала. Ширину также можно регулировать через ручку перетаскивания после размещения. |

<Note>
**Horizontal Line** — единственный инструмент трендовых линий, поддерживающий **ценовые оповещения**. При включении оповещения вы можете настроить условие (пересечение, цена выше или цена ниже) и поведение уведомления (однократное или повторяющееся). Подробности на странице [Оповещения](/advanced/alerts).
</Note>

### 3. Фибоначчи (3 инструмента)

Инструменты на основе чисел Фибоначчи для определения уровней коррекции, целей расширения и институциональных зон ордеров.

| Инструмент | Точки | Описание |
|------------|-------|----------|
| **Fib Retracement** | 2 | Стандартная коррекция Фибоначчи. Размещайте от максимума к минимуму колебания (или наоборот) для отображения ключевых уровней: 0%, 23.6%, 38.2%, 50%, 61.8%, 78.6%, 100%. Можно добавлять пользовательские уровни. |
| **Fib Extension** | 2 | Проецирует уровни Фибоначчи за пределы измеренного движения для определения целей: 127.2%, 141.4%, 161.8%, 200%, 261.8%. |
| **Premium/Discount** | 2 | ICT-стиль зон премиум и дисконт с выделенной зоной оптимального входа (OTE) между 62% и 79%. Отдельные цвета для зоны премиум, дисконт и OTE. Поддерживает продление луча OTE. |

#### Настройка Фибоначчи

| Настройка | Описание |
|-----------|----------|
| **Custom Levels** | Добавление или удаление уровней Фибоначчи (например, 88.6% для гармонических паттернов) |
| **Level Colors** | Назначение индивидуальных цветов каждому уровню |
| **Background Fill** | Заливка зон между уровнями полупрозрачным цветом |
| **Show Labels** | Отображение процентов и ценовых значений на каждом уровне |
| **Show Price Labels** | Отображение рассчитанной цены на каждом уровне Фибоначчи |
| **Reverse** | Перевернуть направление измерения Фибоначчи |
| **Extend Left/Right** | Продлить линии уровней за пределы точек привязки |

### 4. Ганн (2 инструмента)

Инструменты геометрического анализа У.Д. Ганна для изучения соотношений цены и времени.

| Инструмент | Точки | Описание |
|------------|-------|----------|
| **Gann Box** | 2 | Сетка цена-время, разделяющая выбранный диапазон на пропорциональные секции на основе углов Ганна (1x1, 1x2, 2x1 и т.д.). Полезен для определения симметрии цена-время и потенциальных точек разворота. |
| **Gann Fan** | 2 | Веер линий углов, расходящихся из опорной точки под стандартными углами Ганна. Линия 1x1 (45 градусов при правильном масштабе графика) — это угол равновесия. Цена выше 1x1 — бычья; ниже — медвежья. |

| **Rotated Rectangle** | 3 | Прямоугольник с возможностью поворота на любой угол. Первые две точки определяют одну сторону; третья задаёт ширину. |

| **Ellipse** | 2 | Настраиваемый овал, определяемый ограничивающим прямоугольником. Полезен для выделения округлых зон. |
| **Arc** | 2 | Сегмент изогнутой дуги. Настраиваемые начальный и конечный углы. |
| **Path** | Много | Произвольный контур с несколькими контрольными точками. Нажмите для добавления точек; двойной клик для завершения. |
| **Polyline** | Много | Многосегментная соединённая линия. Каждый клик добавляет вершину; двойной клик для завершения. |

| **Brush** | Свободно | Рисование от руки, следующее за движением мыши. Применяются ширина и цвет обводки. |
| **Highlighter** | Свободно | Полупрозрачное наложение от руки. Аналогично кисти, но с принудительно низкой прозрачностью для эффекта маркера. |

### 6. Текст и аннотации (3 инструмента)

### 7. Графические паттерны (11 инструментов)

Готовые шаблоны для классических графических паттернов и моделей рыночной структуры. Каждый паттерн предоставляет размеченные точки привязки, которые вы размещаете на графике.

| Инструмент | Описание |
|------------|----------|
| **Head and Shoulders** | Паттерн разворота из **7 точек** (Start→LS→Valley(L)→Head→Valley(R)→RS→End). Лейблы LS/Head/RS автоматически размещаются на ключевых точках. Линия шеи рисуется от края до края паттерна. Автофиниш при наборе 7 точек. |
| **XABCD (Harmonic)** | Гармонический паттерн из **5 точек** (X, A, B, C, D). Кружки-якоря на каждой точке, лейблы смещаются вверх/вниз автоматически. Автофиниш при наборе 5 точек. |
| **Elliott Wave** | Импульсная волна из **6 точек** (0→1→2→3→4→5). Кружки-якоря и стандартные лейблы волн. Автофиниш при наборе 6 точек. |
| **Quasimodo Bull** | Шаблон бычьего разворотного паттерна QM. Обозначает ключевые структурные точки паттерна. |
| **Quasimodo Bear** | Шаблон медвежьего разворотного паттерна QM. |
| **Wyckoff Accumulation #1** | Схематическая структура накопления (модель Spring). Включает размеченные события: PS, SC, AR, ST, Spring, Test, SOS, LPS, BU. |
| **Wyckoff Accumulation #2** | Альтернативная схема накопления с другим структурным акцентом. |
| **Wyckoff Distribution #1** | Схематическая структура распределения (модель UTAD). Включает размеченные события: PSY, BC, AR, ST, UT, UTAD, LPSY, SOW. |
| **Wyckoff Distribution #2** | Альтернативная схема распределения. |
| **Wyckoff Re-accumulation** | Паттерн продолжения накопления, формирующийся внутри существующего восходящего тренда. |
| **Wyckoff Re-distribution** | Паттерн продолжения распределения, формирующийся внутри существующего нисходящего тренда. |

#### Настройки паттернов Вайкоффа

| Настройка | Описание |
|-----------|----------|
| **Hide Line** | Скрыть зигзагообразную траекторию, показывая только метки событий |
| **Hidden Events** | Выборочно скрыть отдельные метки событий (например, скрыть «AR» и «ST» для уменьшения загромождённости) |

### 8. Торговые позиции (2 инструмента)

Визуальные наложения риск/прибыль для планирования сделок.

| Инструмент | Точки | Описание |
|------------|-------|----------|
| **Long Position** | 3 | Отображает цену входа, тейк-профит (TP) и стоп-лосс (SL) для длинной сделки. Показывает соотношение риск/прибыль, потенциальную прибыль и потенциальный убыток в долларах. |
| **Short Position** | 3 | Аналогично Long Position, но для коротких сделок. Вход выше, TP ниже, SL выше. |

#### Настройки калькулятора позиции

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|
| **Account Size** | Баланс торгового счёта для расчёта P&L | 10 000 |
| **Lot Size** | Размер позиции в контрактах или единицах | 1 |
| **Tick Value** | Стоимость одного тика в долларах | Авто |
| **Risk Opacity** | Прозрачность зоны стоп-лосса | 0.2 |
| **Reward Opacity** | Прозрачность зоны тейк-профита | 0.2 |
| **Show Stats** | Отображать числа P&L и R:R на рисунке | Вкл |

Инструмент позиции автоматически рассчитывает и отображает:

- Риск в долларах (вход до SL * размер лота * стоимость тика)
- Прибыль в долларах (вход до TP * размер лота * стоимость тика)
- Соотношение риск/прибыль
- Процент счёта под риском

<Tip>
Используйте инструмент позиции перед каждой сделкой для визуального подтверждения, что ваше соотношение риск/прибыль соответствует минимальному порогу. Распространённое правило — требовать не менее 2:1 прибыли к риску перед входом в сделку.
</Tip>

### 9. Прогноз и анализ (11 инструментов)

Инструменты для проекции, измерения и расширенных аналитических наложений.

| Инструмент | Точки | Описание |
|------------|-------|----------|
| **Forecast** | 2 | Маркер направленного прогноза, показывающий ожидаемое ценовое движение за временной период. |
| **Bars Pattern** | Много | Копирование исторического ценового паттерна и проекция вперёд. Выделите диапазон свечей, затем разместите проекцию по текущей цене. |
| **Ghost Feed** | Много | Полупрозрачное наложение прошлой ценовой последовательности, проецированной вперёд. Аналогично Bars Pattern, но отображается как полупрозрачные свечи. |
| **Projection** | 3 | Проекция измеренного движения. Разместите три точки (A-B-C) — инструмент спроецирует точку D, где измеренное движение завершается. |
| **Anchored VWAP** | 1 | Средневзвешенная по объёму цена, рассчитанная от пользовательской точки привязки. Перетащите привязку на любую свечу для расчёта VWAP от этой точки вперёд. |
| **Fixed Range Volume Profile** | 2 | Гистограмма распределения объёма для выбранного временного диапазона. Показывает объём на каждом ценовом уровне внутри выбранного периода. |

| **Price Range** | 2 | Измерение вертикального расстояния между двумя ценовыми уровнями. Отображает разницу в цене, процентах и тиках. |
| **Date Range** | 2 | Измерение горизонтального времени между двумя точками. Отображает продолжительность в барах и календарном времени. |
| **Date and Price Range** | 2 | Комбинированное измерительное поле, показывающее и ценовое расстояние, и временную продолжительность. |

#### Настройки рисунков профиля объёма

## Работа с рисунками

### Выделение и перемещение

Нажмите на любой рисунок стандартным курсором для его выделения. Выделенные рисунки отображают ручки привязки в контрольных точках. Перетаскивайте ручки для изменения размера или формы; перетаскивайте тело рисунка для перемещения.

### Множественное выделение

Удерживайте `Shift` и нажимайте на несколько рисунков для их выделения как группы. Затем можно перемещать всю группу вместе или удалить все сразу.

### Шаблоны стилей

Сохраните визуальные свойства рисунка как именованный шаблон стиля. Применяйте шаблон к другим рисункам того же типа для единообразного визуального оформления графика. Шаблоны сохраняются между рабочими пространствами и включаются в облачную синхронизацию.

### Сохранение

Все рисунки автоматически сохраняются как часть вашего рабочего пространства. Они сохраняются при перезапуске приложения и включаются в облачную синхронизацию, если ваша лицензия это поддерживает. Переключение символов или таймфреймов не удаляет рисунки — они остаются привязанными к своим координатам цена/время.
