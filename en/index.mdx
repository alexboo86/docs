<Warning>
**Alpha Version** — Cluster Terminal is currently in alpha. The product is provided "as is" without warranty. Features may change, and you may encounter bugs. Please report any issues via [Telegram](https://t.me/alexboo86) or the [feedback page](https://tally.so/r/jaM4o1).

Cluster Terminal is a professional desktop application for cryptocurrency futures analysis. It combines cluster (footprint) charts, over 50 order flow indicators, and real-time market data in a fast, WebGL-powered interface built for serious traders.

## Key Highlights

<Steps>
  <Step title="Real-Time Streaming">
    WebSocket-powered live candles, clusters, order book, and open interest from Binance Futures & Spot. A background integrity checker ensures your data stays accurate at all times.
  </Step>
  <Step title="Cluster (Footprint) Charts">
    Volume-at-price visualization with delta coloring, configurable aggregation, and cloud overlays across all timeframes from 1 minute to 1 month.
  </Step>
  <Step title="Order Flow Indicators">
    Over 50 indicators purpose-built for futures: absorption alpha, CVD divergence, stacked imbalances, liquidation levels, order blocks, fair value gaps, market structure, and more.
  </Step>
  <Step title="High Performance">
    SciChart WebGL rendering handles millions of data points smoothly. Progressive loading keeps the UI responsive even with deep history going back up to 365 days.
  </Step>
</Steps>

<Note>
  Cluster Terminal currently supports **Binance Futures (USDT-M perpetual contracts) and Spot markets**. Additional exchanges (Bybit, OKX) are planned for future releases.
</Note>

## Data Coverage

| Data Type | Source | History Depth |
|-----------|--------|---------------|
| Candles (OHLCV) | Binance Futures & Spot | Up to 365 days (or from listing date if sooner) |
| Cluster / Footprint | Aggregated trades (Futures & Spot) | Up to 365 days (or from listing date if sooner) |
| Open Interest (OHLC) | Binance Futures | Up to 365 days (or from listing date if sooner) |
| Order Book (DOM) | Binance Futures & Spot | Real-time only |
| Liquidations | Binance Futures | Real-time only |
| Funding Rate | Binance Futures | Real-time only |
| Long/Short Ratio | Binance Futures | Real-time only |
