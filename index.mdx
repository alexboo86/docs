Cluster Terminal — это профессиональное десктопное приложение для анализа криптовалютных фьючерсов. Оно объединяет кластерные (footprint) графики, более 50 индикаторов потока ордеров и рыночные данные в реальном времени в быстром интерфейсе на основе WebGL, созданном для серьёзных трейдеров.

## Ключевые особенности

<Steps>
  <Step title="Потоковые данные в реальном времени">
    Живые свечи, кластеры, стакан и открытый интерес через WebSocket с Binance Futures & Spot. Фоновый контроль целостности данных обеспечивает точность информации в любое время.
  </Step>
  <Step title="Кластерные (Footprint) графики">
    Визуализация объёма по цене с раскраской по дельте, настраиваемой агрегацией и облачными оверлеями по всем таймфреймам от 1 минуты до 1 месяца.
  </Step>
  <Step title="Индикаторы потока ордеров">
    Более 50 индикаторов, специально разработанных для фьючерсов: абсорбция, дивергенция CVD, стековые дисбалансы, уровни ликвидаций, блоки ордеров, разрывы справедливой стоимости, рыночная структура и другие.
  </Step>
  <Step title="Высокая производительность">
    Рендеринг SciChart на WebGL обрабатывает миллионы точек данных без задержек. Прогрессивная загрузка сохраняет отзывчивость интерфейса даже при глубокой истории до 365 дней.
  </Step>
</Steps>

<Note>
  Cluster Terminal в настоящее время поддерживает **Binance Futures (бессрочные контракты USDT-M) и Spot-рынки**. Поддержка дополнительных бирж (Bybit, OKX) запланирована в будущих релизах.
</Note>

## Покрытие данных

| Тип данных | Источник | Глубина истории |
|-----------|--------|---------------|
| Свечи (OHLCV) | Binance Futures & Spot | До 365 дней (или с даты листинга, если раньше) |
| Кластеры / Footprint | Агрегированные сделки (Futures & Spot) | До 365 дней (или с даты листинга, если раньше) |
| Открытый интерес (OHLC) | Binance Futures | До 365 дней (или с даты листинга, если раньше) |
| Стакан (DOM) | Binance Futures & Spot | Только в реальном времени |
| Ликвидации | Binance Futures | Только в реальном времени |
| Ставка финансирования | Binance Futures | Только в реальном времени |
| Соотношение Long/Short | Binance Futures | Только в реальном времени |
