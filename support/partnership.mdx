# Сотрудничество

Cluster Terminal — активно развивающаяся профессиональная торговая платформа. Мы открыты к сотрудничеству и ищем партнёров для развития проекта.

## Инвестиционные возможности

Мы ищем инвесторов для:

- **Инфраструктура** — содержание и масштабирование серверов данных в нескольких регионах для низкой задержки по всему миру
- **Разработка** — расширение команды для ускорения разработки, добавления новых бирж и улучшения платформы
- **Покрытие данных** — добавление поддержки новых бирж (Bybit, OKX, dYdX) и классов активов

## Почему Cluster Terminal

- Профессиональный инструмент кластерного анализа, построенный с нуля на современных технологиях (Tauri, React, WebGL, Rust)
- Более 50 индикаторов потока ордеров, 62 инструмента рисования, потоковые данные в реальном времени
- Растущая база пользователей в крипто-трейдинг сообществе СНГ и Европы
- Полностью собственная инфраструктура с низкими операционными затратами
- Активная разработка с регулярными релизами
